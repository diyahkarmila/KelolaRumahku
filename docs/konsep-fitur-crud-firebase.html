Konsep

KelolaRumahKu

KelolaRumahKu adalah website manajemen rumah tangga berbasis web untuk mengelola daftar belanja, pengeluaran, peralatan rumah, kategori, dan aktivitas rumah tangga dalam satu sistem terintegrasi.

Fitur Halaman

Halaman Utama

  • Dashboard: ringkasan statistik, pengeluaran, daftar belanja aktif, aktivitas terakhir
  • Daftar Belanja: tambah, lihat, edit, hapus item belanja
  • Pengeluaran: tambah, lihat, edit, hapus data pengeluaran
  • Peralatan Rumah: tambah, lihat, edit, hapus inventaris rumah
  • Kategori: tambah, lihat, edit, hapus kategori
  • Aktivitas: melihat riwayat aktivitas rumah tangga
  • Login & Register: autentikasi user
  • Admin: monitoring data global
Pembagian 12 CRUD

CRUD Inti Sistem

  1. Create daftar belanja
  2. Read daftar belanja
  3. Update daftar belanja
  4. Delete daftar belanja
  5. Create pengeluaran
  6. Read pengeluaran
  7. Update pengeluaran
  8. Delete pengeluaran
  9. Create peralatan rumah
  10. Update/Delete peralatan rumah (manajemen inventaris)
  11. CRUD kategori
  12. Read aktivitas rumah tangga

Alur Utama

Flow Sistem

User register/login → menambahkan daftar belanja → menandai barang sudah dibeli → data otomatis masuk ke pengeluaran → jika barang berupa aset, masuk ke peralatan rumah → user/admin dapat mengelola, mengupdate, atau menghapus data → seluruh aktivitas tercatat dalam riwayat rumah tangga.
